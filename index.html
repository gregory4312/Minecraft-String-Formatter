How to Use This Tool

Getting Started

This tool helps you easily format strings and create bindings for Minecraft JSON UI. Here's how to use it:

1. Set Up Your Flag

Enter the flag pattern that appears at the beginning of your string (e.g., "§f§l§a§g") or leave empty if you don't use a flag.

2. Input Your String

Paste the string you want to format. The tool will automatically remove the flag (if any) and show you the cleaned text.

3. Select Text Ranges

  1. Click "Select Range" to enter selection mode
  2. Click on the start position of the text you want to bind
  3. Click on the end position to complete the selection
  4. Characters will blink when you're in selection mode

4. Create Bindings

After selecting a range, enter a target property name (like "#player_health") and click "Create Binding".

5. Import/Export

You can import existing bindings or export your created bindings as JSON.

Try the Template

Click "Load Template" to see an example with a sample string that demonstrates common binding patterns.
